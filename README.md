# th06_2.0
东方project自制小demo，殴打蕾咪

<Br/>【基本操作】
<Br/>  方向键移动
<Br/>  shift减速聚焦火力
<Br/>  Z攻击
<Br/>  X大招（二妹）

<Br/>【内容】
<Br/>  有完整的游戏流程
<Br/>  敌人会掉落各种奖励，拾取可以获得对应的加成
<Br/>  有BGM，可以流程的切换
<Br/>  有终符，坚持20秒
